# 🏗️ Construction Company Website

A dynamic and responsive full-stack web application for a construction company, built with **React** (frontend) and **Laravel** (backend). Includes an admin panel for managing content such as projects, services, testimonials, and blogs.

---

## 🚀 Features

- ✅ Fully responsive UI
- ✅ Admin panel with full **CRUD** operations:
  - Projects
  - Services
  - Testimonials
  - Blogs
- ✅ RESTful API integration
- ✅ Clean and professional design
- ✅ Easy content management

---

## 🛠️ Tech Stack

### Frontend

- **React Router**
- **Tailwind CSS** (if used)

### Backend
